# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## 1.0.3 - 2026-02-24

### Added

- Python Hexa template support
- Implement python deps installation logic
- Changelog
- Updated README.md

## [1.0.2][1.0.2] - 2026-02-23

### Changed

- Updated AiMind Next.js template

## [1.0.1][1.0.1] - 2026-02-20

### Fixed

- Fixed path issues when moving core logic into src directory
- Template directory now properly references src/templates

## [1.0.0][1.0.0] - 2026-02-18

### Added

- Initial release with template scaffolding CLI
- Interactive template selection
- Automatic git initialization
- Automatic dependency installation
- Support for tar.gz template archives
- Template metadata system
- Colorful console output with progress indicators

### Changed

- Convert to ES modules for inquirer compatibility
